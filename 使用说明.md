# 使用说明 (Usage Guide)

## 📋 系统要求

| 要求 | 说明 |
|------|------|
| 📱 设备 | Android 设备 |
| 🎮 游戏 | 第五人格 (Identity V) 所有版本 |
| 🔓 权限 | **Root 权限** (必需) |

---

## 🚀 使用步骤

### 1. 放置文件

将程序文件放置到 `/data` 目录下

### 2. 选择驱动

从 `RT-devpro+` 压缩包中选择适配你设备的驱动

### 3. 设置权限

给予 777 权限(驱动和Identity5.sh均需要)

### 4. 刷入驱动

**先刷入驱动**，确保驱动加载成功

### 5. 启动游戏

打开第五人格游戏

> ⚠️ **注意**: 必须先打开游戏再执行程序，否则会空载！

### 6. 执行程序

勾选使用ROOT权限执行Identity5.sh

### 7. 唤出悬浮窗

使用 **音量键** 唤出/隐藏悬浮窗

| 按键 | 功能 |
|------|------|
| 音量+ | 显示悬浮窗 |
| 音量- | 隐藏悬浮窗 |

---

## ⚠️ 注意事项

- ✅ 确保设备已获取 Root 权限
- ✅ 确保驱动与设备匹配
- ✅ 必须先启动游戏再运行程序
- ❌ 游戏未启动时运行会导致空载

---

## 📁 目录结构

```
/data/
├── <程序源码>
└──Identity5.sh (编译出的可执行文件)
└── RT-devpro+/
    └── <驱动文件>
```

---

## 🔧 常见问题

### Q: 程序无响应？
A: 检查是否先启动了游戏，未启动会空载。

### Q: 悬浮窗不显示？
A: 尝试按音量+键唤出。

### Q: 权限不足？
A: 确保已执行 `chmod 777` 并具有 Root 权限。
